# BFF with Apollo

## Development

### Prerequisites

- Node 18
- _Rust?_

### Getting started

#### 1. Starting the APIs to integrate

```bash
npm run start:integrations
```

#### 2. Starting the supergraph

```bash
npm run start:gateway
```
